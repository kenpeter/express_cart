## Intro

nodejs and express shopping cart based on https://www.youtube.com/watch?v=-3vvxn78MH4&index=2&list=PL55RiY5tL51rajp7Xr_zk-fCFtzdlGKUp


## Install
npm install


## Run
* node ./seed/product-seeder.js
* npm start
